Qu'est-ce que la Wild Code School ?
Promotion JAVA / JEE / Spring | Mai 2019

La Wild Code School est une école proposant une formation de développeur web en 5 mois et 4 mois de stage en entreprise. Sur Reims, la formation est axée sur JAVA / JEE / SPRING, elle se trouve au Quartier Libre, un espace de coworking moderne où diverses startups se cotoient.Ce site met en avant les étudiants (wilders), de la promotion "ECHO" de la ville champenoise. N'hésitez pas à découvrir leur personnalité et les contacter en cliquant sur leur profil.
